### 🎮 Game Backlog — Gerenciador de Jogos
> Aplicação web full-stack para gerenciar biblioteca pessoal de jogos. Permite buscar títulos via RAWG API, adicioná-los à coleção e acompanhar o progresso com status, avaliação de 1–5 estrelas e notas pessoais. Interface dark minimalista com ordenação por nome, status ou avaliação.

**Tecnologias:** Node.js · Express · PostgreSQL · EJS · Axios · CSS3 · RAWG API · Render

**Funcionalidades:**
- 🔍 Busca de jogos pelo nome via RAWG API
- ➕ Adição de jogos à biblioteca pessoal
- 📊 Gerenciamento de status (Quero jogar, Jogando, Zerado, Abandonado)
- ⭐ Avaliação de 1 a 5 estrelas e notas pessoais
- 🔃 Ordenação por nome, status ou avaliação
- ✏️ Edição e remoção de jogos
- 💾 Persistência de dados em PostgreSQL
- 🚀 Deploy em produção no Render

[![Repositório](https://img.shields.io/badge/Ver%20Repositório-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/GuilhermeAlves58/Game-Backlog)
[![Demo](https://img.shields.io/badge/Demo%20ao%20Vivo-00C853?style=for-the-badge&logo=render&logoColor=white)](https://game-backlog-zy2v.onrender.com/)
